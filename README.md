# STYLiSH

[STYLiSH](https://stylish-cbc00.web.app/homePage.html) 服裝銷售電子商務網站 

## 主要功能  
* 瀏覽、搜索產品
* 查看產品詳細資訊
* 將產品添加到購物車
* 購物車結帳
* Facebook 第三方登入
  
## 開發技術   
&ensp;**JavaScript**  
 * 完成首頁的邏輯與互動事件  
 * 完成輪播圖及無限滾動  
  
&ensp;**React**  
 * 完成商品頁、購物車頁  
 * 使用 useState 管理狀態  
  
&ensp;**Styled Components：**  透過 Styled Components 完成樣式設計及響應式  
  
&ensp;**Tap Pay：**   金流服務
  
&ensp;**Facebook SDK：**  第三方會員登入
